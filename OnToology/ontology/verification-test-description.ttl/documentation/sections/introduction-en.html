Introduction back to ToC

Ontology verification is the ontology evaluation which compares the ontology against the ontology specification document. To carry out this ontology verification a set of test cases should be generated. This cases cases should be stored, shared and published in order to allow reuse between users and ontologies. Moreover, the test cases should be able to provide traceability between the requirements from which they are extracted and the test cases. This document describes how test cases oriented to ontology verification can be defined.
